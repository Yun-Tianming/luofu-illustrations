# Prompt Examples

## 只做配图规划

```text
Use $luofu-illustrations 先不要生图。
分析下面文章的认知锚点，输出约 5 张 shot list。
每张写清：位置、主题、核心意思、结构类型、选角、选角理由、角色动作、建议元素和中文标注。

<粘贴文章>
```

## 直接生成正文配图

```text
Use $luofu-illustrations 为下面文章生成 4 张正文配图。
根据内容自动选择韩立、南宫婉或双角色；不要为了情侣感强行同框。
要求：16:9、纯白背景、蓝黑手绘线稿、蓝金角色配色、少量中文手写标注。

<粘贴文章>
```

## 韩立：风险与验证

```text
Use $luofu-illustrations 为“做决定前先验证最危险的假设”生成一张图。
使用韩立。他必须通过检查、试探或预留后手来表达谨慎，而不是站着皱眉。
```

## 南宫婉：探索与连接

```text
Use $luofu-illustrations 为“让分散的灵感彼此连接，形成新方向”生成一张图。
使用南宫婉。让她通过牵引、点亮或打开结构表达灵动与行动力，不要只做可爱姿势。
```

## 双角色：发散与收敛

```text
Use $luofu-illustrations 为“先大胆发散，再谨慎收敛”生成一张图。
南宫婉负责展开可能性，韩立负责筛选和校准；两人的动作必须形成因果关系。
不要拥抱、牵手、相望或添加粉色恋爱气氛。
```

## 改图：角色纠错

```text
Use $luofu-illustrations 修正这张图里的韩立。
保持核心构图和其他元素不变，严格参照 Skill 内的韩立角色设定图修正发型、金簪、深蓝金边外袍和浅蓝内袍。
不要生成角色设定表或多个角度。
```

## 改图：去掉标题

```text
Use $luofu-illustrations 去掉这张图左上角的“流程图”及下划线。
角色、标注、构图和手绘质感保持不变，不要新增任何内容。
```
